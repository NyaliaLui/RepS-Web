How to use RepS

  1. Put all your replays into a .zip file
  2. Upload the .zip file you just created at sc2RepS.com
  3. Select to either Sort by player name or SC2 matchup
  4. Wait a few seconds for download link
  5. Click the download link and your organized replays will be downloaded as the file Replays.zip

Notes and Limitations

  • sort by player name will result in the same replay appearing in both player's folders.
  • sort by player name is case sensitive, so a player name NoticALs and noticals will have their own folders of replays.
  • sort by SC2 matchup currently does not work for team games.
  • RepS does not convert player names or race names to different languages. So, if a player's name is in Korean, then their name will show up in Korean. The same applies to SC2 race.
  • Currently, running RepS on a zip archive which RepS generated will either create duplicate replays or have undefined behaviour.
